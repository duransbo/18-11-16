About

The interdisciplinary project on the PhoneGap development tool aims to describe the use of technology tools to develop applications for mobile devices through structured web views in HTML, CSS and JavaScript in order to assist researchers from different areas of knowledge in the collection data for their research, whether for some kind of practical solution or theoretical basis for some future research.

The project features a website that shows the basic operation of the PhoneGap tool with a video showing the installation and development of a simple application plus a tutorial explaining in more detail the video content and a FAQ session with some frequently asked questions from users / developers.

This project is related disciplines of English Instrumental, Human Computer Interface and Introduction to Research Methodology and was initially conceived by the teachers Gustavo Prieto and Rosana Ferrareto.

FAQ

What is phonegap for?

To create softwares that can be use in differents devices in a fast and easy way.

Why use phonegap?

You can use it to create softwares that can do whatever you want for example, data collect, inventory control, files sharing, etc.

What do you need to use PhoneGap?

It is only necessary a computer and an internet connection for you to create your only application. After that you can use it in any mobile device accessing where the app is stored.

How to install Phonegap?

You can check our installation tutorial a our site or check the documentation at docs.phonegap.com.
